# Overview
This is a collection of user-made scripts for the [FreeSpace Open engine](https://github.com/scp-fs2open/fs2open.github.com)

## List of Scripts
|Name|Description|min. FSO|Dependencies|Scripter|
|---|---|---|---|---|
